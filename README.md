### Business Problem Statement
The objective of this project is to analyze IT incident data to identify SLA breaches, understand patterns by priority and assignment group, and provide actionable insights to improve IT service performance.

# Tools Used
| Tool         | Why Companies Use It       |
| ------------ | -------------------------- |
| ServiceNow   | IT Incident Management     |
| Python       | Data analysis & automation |
| Pandas       | Data cleaning              |
| NumPy        | Calculations               |
| SQL          | Querying incident data     |
| Seaborn      | Reports & dashboards       |
| Google Colab | Cloud-based analysis       |

*Final Business Insights*
Key Findings:

SLA breaches are highest in Network team
P1 incidents show unexpected breaches
SLA adherence needs improvement in high-impact incidents

Recommendations:
Improve escalation for P1 incidents
Review workload distribution
Strengthen monitoring & automation in ServiceNow
